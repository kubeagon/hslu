# Datentypen, Operatoren & Ausdrücke

## Zeichenvorrat Java

- ASCII und ISO-Latin-1 (ISO 8859-1)
- Ziffern `0 - 9`
- Leerzeichen
- Steuerzeichen `\n, \t`
- Sonderzeichen:
    - Satzzeichen: `(), {}, [], ., ;`
    - Operatoren: `+, -, /, *, <, !=`
    - `',"`
- Zeichensatz ist Unicode
- Codierung von Zeichen ist UTF-8 das heisst 2 oder 4 Bytes pro Zeichen

## Kommentare

- Einzeilige Kommentare: `//`
- Mehrzeilige Kommentare `/*    */`
- Dokumentationskommentare die automatisch zu HTML Dateien verwandelt werden: `/**    */`

## Reservierte Wörter

class, public, int, double , import, return, for, etc…

## Namungsgebung

CamelCase

## Konstanten
 
Ändert ihren Wert nicht.

Beispiel Konstante:

```java
final int MINIMUM = 10;
```

Beispiel literale Konstante:

```java
double endGuthaben = 12860.55;
```
