## Question 16

![alt text](question16.png)

- logging console → sends log/debug messages only to the console port (not to VTY).
- terminal monitor → enables debug/log messages to appear on VTY (SSH/Telnet) sessions.
- logging buffered → stores logs in RAM.
- logging synchronous → prevents logs from interrupting your typing.

## Question 20

![alt text](question20.png)

| Symbol | Meaning                                                       |
| ------ | ------------------------------------------------------------- |
| **!**  | Successful ping (received ICMP echo reply)                    |
| **.**  | Timeout (no reply received)                                   |
| **U**  | ICMP *Destination Unreachable* received from a router or host |
| **Q**  | Source quench (rare, deprecated)                              |
| **M**  | Could not fragment packet                                     |
| **?**  | Unknown packet type                                           |
