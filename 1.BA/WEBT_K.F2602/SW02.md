# HTML

HTML (HyperText Markup Language) is the standard declarative language used to structure content on the web. It defines the elements of a webpage and uses tags (e.g., `<p>`, `<h1>`, `<a>`) to describe the structure and meaning of content, while the browser interprets these tags and displays the page.

Here's an example HTML document

```html
<html>
    <head>
        <title>Hello World</title>
    </head>
    <body>
        <p>Hallo WorldWideWeb!</p>
    </body>
</html>
```

## `<head>` element

The information within the head section are not displayed in the document in your browser. They contain things such as:

- Title `<title>`
- Metadata `<meta>`
- Style `<style>`
- Script definitions `<script>`

## `<body>` element

The body tag marks the beginning of the displayed part of a HTML document. Each document can only contain one body section.

## Formating Tags

- Paragraph: `<p> ... </p>`
- Newline: `<br/>`
- Preformatted: `<pre> .. </pre>`
- Headings: `<h1> ... </h1>` up to `<h6> ... </h6>` 
- Seperator line: `<hr/>`
- Container: `<div> ... </div>` or `<span> ... </span>`

## Inserting graphics

HTML supports many different kinds of formats, such as gif, jpg, png, svg, webp, etc. The `alt` attribute is for accomadating the blind and searchengines.

```html
<img src="path/to/image>" alt="description" />
```

## Hyperlinks

Open another web-address when used.

- `_self` is the default and tells the browser to use the existing tab
- `_blank` tells the browser to open a new tab

**Absolut** zur Referenzierung von Ressourcen in anderer Domain als die aktuell angezeigten HTML-Dokuments

```html
<a href="https://page.example.com" target="_blank">example text to display</a>
```

**Relativ** zur URL des aktuell angezeigten Dokuments zur Referenzierung von Ressourcen mit gleicher Domain wie aktuelles HTML-Dokument

Annahme die aktuelle URL ist: `https://example.com/news/2024/artikel123.html`

```html
<a href="/2025/artikel223.html">example test to display</a>
```

Dies resultiert in folgende Addresse: `https://example.com/news/2025/artikel223.html`

## Internal Links

Reference a heading within the same HTML document.

```html
<a href="#Chaptername">Chapter name</a>

<p> id="Chaptername">Chapter name</p>
```

## Metatags

Werden nicht zur Gestaltung, sondern Beschreibung des Inhaltes verwendet. Sie werden im head-Bereich eingefügt. 

```html
<meta name="Schlüsselwort" content="Inhalt">
```

Die verschiedenen Metatags könne hier eingesehen werden: [Metatags Whatwg.org](https://html.spec.whatwg.org/multipage/semantics.html#the-meta-element)

## Entities

Zur konkreten Darstellung von Sonderzeichen muss das charset Metatag korrekt gesetzt sein

```html
<meta charset="utf8">
<meta charset="iso-8859-1">
```

Escapes sind notwendig bei Zeichen, welche mit dem HTML-Markup kollidieren. Beispiele hier: [htmlhelp escapes](http://www.htmlhelp.com/reference/html40/entities/latin1.html)

## Masseinheiten

Werden verwendet um die Dimensionen verschiedener Elemente we Bilder, Schriften, Ränder, Abstände, etc zu bestimmen. Heutzutage werden vorallem relative Masseinheiten verwendet.

### Relative Einheiten

- px (Pixel): Abhängig der Pixeldichte des Ausgabegeräts
- em: Relativ zur Schriftgrösse des Elements M
- % (Prozent): Typischerweise relativ zum Elternelement

### Absolute Einheiten

- mm: Millimeter
- in: Inch
- pt: Punkt

## Farben

Farben werden aus den RGB Wertangaben gebildet

- Werte werden in HEX angegeben
- Pro Farbe sind Werte von #00 bis #FF möglich
- Einige Farben sind per Namen vordefiniert

## Strukturierung per sematischer Container

### `<main>`-Element

Inhalt welcher dem zentralen Thema eines Dokuments oder der zentralen Funktionalität einer Applikation entspricht

- Es darf exakt einmal in einem HTML-Dokument vorkommen

### `<header>`-Element

Enthält den sichtbaren Kopfbereich eines HTML-Dokuments

```html
<header>
    <h1>HTML5 Seminar</h1>
</header>
```

### `<footer>`-Element

Enthält Informationen die in HTML-Dokumenten am Ende stehen

- Ihre Position ist nicht Notwendigerweise am unteren Rand

```html
<footer>
    <p>macro</p>
    <p>CC 3.0 Lizenz</p>
</footer>
```

### `<aritcle>`-Element

Stellt eine in sich geschlossene Abschnitt eines Dokuments dar

- Innerhalb von article-Elementen könne header, sections und footers beinhaltet sein

### `<section>`-Element

Enthalten eine thematische Gruppierung von Inhalten typischerweise mit einer Überschrift

- Dienen dazu den Inhalt einer article in semantsiche Abschnitte zu gliedern

### `<nav>`-Element

Umschliesst insbesondere Navigationsleisten und kann neben  Verweisen auch eine Überschrift oder ähnliches enthalten

```html
<nav>
    <ul>Hauptnavigation
        <li>Startseite</li>
    </ul>
</nav>
```

### `<aside>`-Element

Umschliesst Abschnitte einer Seite, deren Inhalt nur in einem inderekten Zusammenhang mit dm umgebenden Inhalt stehen

```html
<aside>
    <p>Weiterführende Infos</p>
</aside>
```

## Formulare

Formulare ermöglichen es Benutzer zwischen Optionen zu wählen oder Date wie Adressen, Termine und Texte einzugeben

- Werden mit dem `<form>`-Tag erzeugt
- Bestehen aus Formularelementen

```html
<form>
    <h2>Namenseingabe</h2>
    <label for="vorname">Vorname</label>
    <input type="text" id="vorname">

    <label for="nachname">Nachname</label>
    <input type="text" id="nachname">

    <label for="male">Mann</label>
    <input type="radio" name="gender" id="male">

    <label for="female">Frau</label>
    <input type="radio" name="gender" id="female">

    <label for="familienstand">Familienstand</label>
    <select id="familianstand">
        <option value="ledig">Ledig</option>
        <option value="verheiratet">Verheiratet</option>
    </select>

    <button>Eingaben absenden</button>
</form>
```

- Platzhalter für Eingabevorschläge werden folgendermassen erstellt

```html
<input type="text" id="i" placeholder="example">
```

## Validierung mittels HTML

Clientseitige Eingabevalidierungen können direkt mittels HTML-Attributen spezifiziert werden

- required-Attribut `<input type="text" id="name" required>`
- Formatprüfung `<input type="email" id="mail">`
- Min und Max `<input type="number" id="e" min="5" max="20">`

## Videoelemente

Einbindung von Videos in HTML-Dokumente

```html
<video src="movie.mp4" controls width="360" height="240" poster="startbild.jpg"></video>
```

Beispiel mit mehreren Formate (Browser zeigt das erste Video dessen Format unstertützt ist)

```html
<video controls width="360" height="240" poster="startbild.jpg">
    <source src="movie.ogv" type="video/ogg">
    <source src="movie.mp4" type="video/mp4">
</video>
```
