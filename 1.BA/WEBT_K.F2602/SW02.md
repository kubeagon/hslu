# HTML

HTML (HyperText Markup Language) is the standard declarative language used to structure content on the web. It defines the elements of a webpage and uses tags (e.g., `<p>`, `<h1>`, `<a>`) to describe the structure and meaning of content, while the browser interprets these tags and displays the page.

Here's an example HTML document

```html
<html>
    <head>
        <title>Hello World</title>
    </head>
    <body>
        <p>Hallo WorldWideWeb!</p>
    </body>
</html>
```

## `<head>` element

The information within the head section are not displayed in the document in your browser. They contain things such as:

- Title `<title>`
- Metadata `<meta>`
- Style `<style>`
- Script definitions `<script>`

## `<body>` element

The body tag marks the beginning of the displayed part of a HTML document. Each document can only contain one body section.

## Formating Tags

- Paragraph: `<p> ... </p>`
- Newline: `<br/>`
- Preformatted: `<pre> .. </pre>`
- Headings: `<h1> ... </h1>` up to `<h6> ... </h6>` 
- Seperator line: `<hr/>`
- Container: `<div> ... </div>` or `<span> ... </span>`

## Inserting graphics

HTML supports many different kinds of formats, such as gif, jpg, png, svg, webp, etc. The `alt` attribute is for accomadating the blind and searchengines.

```html
<img src="path/to/image>" alt="description" />
```

## Hyperlinks

Open another web-address when used.

- `_self` is the default and tells the browser to use the existing tab
- `_blank` tells the browser to open a new tab

```html
<a href="https://page.example.com" target="_blank">example text to display</a>
```

## Internal Links

Reference a heading within the same HTML document.

```html
<a href="#Chaptername">Chapter name</a>

<p> id="Chaptername">Chapter name</p>
````
