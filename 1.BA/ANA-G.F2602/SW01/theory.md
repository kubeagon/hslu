# Funktionen

## Was sind Funktionen?

Formelle Ausdrucksweise:

$f : x \mapsto y = f(x)$

$D$ (Definitionsbereich) $\mapsto W$ (Wertebereich)

Beispielfunktion:

$f: x \mapsto \frac{1}{x}$

$D = \{x | x \in \mathbb{R} \land x \neq 0 \}$

$W = \{ x \in \mathbb{R} | x \neq 0 \}$
