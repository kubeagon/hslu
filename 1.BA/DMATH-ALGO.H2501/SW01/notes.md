# Elemetare Datentypen 

## Boolean

Bits die den Wert `Wahr` oder `Falsch` sein können.

### Definition Booleans

Ein boolscher Wert ist entweder True oder False. Mit diesen Werten können wir folgende Operatoren rechenen:

NOT (einstelliger Operator): 
- NOT True = False

AND (zweistelliger Operator):  Nur True wenn beide Operanden True
- True AND True = True
- True AND False = False
- False AND False = False

OR (zwistelliger Operator): Nur False wenn beide Operanden False
- True OR True = True
- True OR False = True
- False OR False = False

## Integer

Einfachsten Zahlen der Mathematik → Ganze Zahlen

### Definition Integer

Ein integer Wert ist entweder eine positive gabze Zahl, die Zahl 0 oder eine negative ganze Zahl

Z = {. . . , −4, −3, −2, −1, 0, 1, 2, 3, 4, . . .}

Addition, Subtraktion und Multiplikation sind wohldefiniert. Bei Division gilt eine Besonderheit:

Ganzahldivision // (zweistelliger Operator):
- 8 // 3 = 3 
- 4 // 2 = 2

Rest % (zweistelliger Operator):
- 10 % 3 = 1
- 10 % 7 = 3
- 10 % 5 = 0

## Float

Alle Zahlen bei denen sich die Nachkommastellen periodisch widerholen sind rational und können als Brüche dargestellt werden. Diese nennen wir rationalen Zahlen Q und gehören zu den reellen Zahlen R.
