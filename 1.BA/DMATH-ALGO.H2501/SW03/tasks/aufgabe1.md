# Series 3 Aufgabe 1

Für a, b, c, n ∈ ℕ sei:

**Fermat(a, b, c, n):**  
Die Summe von a<sup>n</sup> und b<sup>n</sup> ist c<sup>n</sup>.

---

## Lösungsweg

a<sup>n</sup> + b<sup>n</sup> = c<sup>n</sup>

Zahlen einsetzen, die die Gleichung erfüllen oder eben nicht:

- Fermat(1, 1, 1, 1) → ❌ falsch  
- Fermat(1, 1, 1, 2) → ❌ falsch  
- Fermat(1, 1, 1, 3) → ❌ falsch  

- Fermat(1, 2, 3, 1) → ✅ wahr  
- Fermat(2, 1, 3, 1) → ✅ wahr  
- Fermat(3, 4, 5, 2) → ✅ wahr
